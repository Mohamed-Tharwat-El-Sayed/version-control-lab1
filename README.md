## Remove dev branch local
```
git branch -d dev

```
## Remove dev branch remotly
```
git push origin :dev

```

## list tags
```
git tag

```

## delete tag local
```
git tag -d v1.7

```

## delete tag remote
```

git push origin --delete v1.7

```
